# Softwaretestingprojects
Manual Testing Projects – Test Plans & Test Cases

This repository contains a collection of manual testing artifacts, including detailed test plans, test scenarios, and test cases for multiple software applications. It demonstrates end-to-end QA documentation practices, covering requirement analysis, test design, test execution, and defect reporting. Ideal for QA portfolios, training, and reference.
